# Tool Discovery Reports

This directory contains weekly reports from the `tool-discovery-agent`.

## Report Format

Each week, the agent generates a report with:
- New CLI tools discovered
- Replacement candidates for outdated tools
- Trending categories and tools
- Recommendations for integration

## Usage

Reports are named `YYYY-MM-DD.md` (e.g., `2025-11-02.md`)

Review the latest report to see:
1. **New Tools** - Modern CLI utilities worth considering
2. **Replacements** - Better alternatives to current tools
3. **Trends** - Popular categories and emerging tools

## Integration

When you find a tool you want to try:
1. Say: "Install [tool-name]"
2. The `new-tool-integrator-agent` handles full setup automatically
3. Tool is integrated with wrapper, tests, and docs

## Archive

Older reports are kept for reference and trend analysis.
